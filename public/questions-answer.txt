1. What is Interpolation?
Answer:-
It is a one-way data binding technique that inserts component data into the template using double curly braces.
Syntax:-
{{ expression }}
Example:-
<p>{{ username }}</p>
Note:- Useful for displaying values; expressions are evaluated in component context.
